Getting Started

  1. Install the mobile app from the store.
  2. Pair MR headset via Bluetooth/Wi-Fi.
  3. Select your LEGO set in the app.

Building with LEGO-MR

Step-by-step guide for using the app and headset:

  1. Open the "Build" mode on mobile app.
  2. Follow on-screen and MR overlays.
  3. Access contextual menus for hints.
